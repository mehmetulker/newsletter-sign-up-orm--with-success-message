# Newsletter Sign-Up Form

A pixel-based implementation of a Figma newsletter sign-up design using semantic HTML, vanilla CSS, and a sprinkle of JavaScript for client-side validation.

## Live Demo

- https://newslettersignupforms.netlify.app/

## Repository

- https://github.com/mehmetulker/newsletter-signup-form-with-success-message.git

## Features

- Translation of the original Figma layout into clean, accessible markup
- Responsive imagery powered by the `<picture>` element to match mobile and desktop breakpoints
- Vanilla CSS styling with custom properties for consistent spacing, colors, and typography
- Email validation and inline error messaging handled via lightweight JavaScript
- MIT-licensed project ready to extend or integrate into larger codebases

## Getting Started

1. Clone the repository: `git clone https://github.com/mehmetulker/newsletter-signup-form-with-success-message.git`
2. Open the project folder in your editor of choice.
3. Launch `index.html` in your browser (double-click the file or use a simple dev server such as `npx serve`).

The form logic runs entirely in the browser, so no additional build steps or dependencies are required.

## Tech Stack

- HTML5
- Vanilla CSS (no frameworks)
- JavaScript (ES6)
- Google Fonts (Roboto)
